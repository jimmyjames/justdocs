Other Niceties

Hugo offers several other nice features to make writing docs easy.
Embedding videos, images, or gists are trivial.

Hugo supports a feature called shortcodes, which allow for adding content that is not simple Markdown text.

Hugo ships with several shortcodes, and custom ones can be created fairly easy. For example, the tagline, tabbed code blocks, and callouts are all custom shortcodes defined in the JustDocs theme.

Embedding videos

Hugo has both a YouTube and Vimeo shortcode to allow for embedding videos right on your page.

Just grabe the ID of the video, and use it like this:

{{< youtube w7Ft2ymGmfc >}}

Embedding gists

If you have source code stored in GitHub gists, you can use the gist shortcode to easily pull that right into your docs.

Usage

You can provide an optional tagline in the front matter. This will render below the document title. You can use HTML in the tagline value as well (useful for line breaks, icons, or text emphasis like <strong>).

Yaml example:

title: Tagline
tagline: <strong>Call to Action.</strong><br/><br/>Every page features an optional Call to Action to give readers a brief summary of what they will learn, and why they should care.
